🚀 CareerCompass AI

Your Personal Career Navigator

Overview

CareerCompass AI is an intelligent mobile application designed to help
users navigate their career paths through AI-powered recommendations,
skill tracking, and job market insights.

Problem Statement

-   Information overload
-   Lack of personalized guidance
-   Difficulty identifying relevant skills
-   Fragmented tools

Solution

-   AI Resume Analysis
-   Job Matching
-   Skill Tracking
-   Mock Interviews
-   Mentorship
-   Portfolio Builder

Tech Stack

-   Flutter (Dart)
-   Firebase / Node.js
-   Firestore / PostgreSQL
-   OpenAI API / Google NLP

Features

-   User Authentication
-   Resume Analysis
-   Job Matching
-   Skill Tracking
-   Mock Interviews

Getting Started

git clone https://github.com/TUHEU/career_navigator.git
cd career_navigator
flutter pub get
flutter run
