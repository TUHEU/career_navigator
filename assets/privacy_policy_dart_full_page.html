We implement industry-standard security practices to protect your data, including:

  • Encryption of data in transit using TLS and at rest using AES-256.
  • Secure authentication, token management, and session expiry controls.
  • Regular security audits, penetration testing, and vulnerability assessments.
  • Strict internal access controls — only authorized personnel can access personal data.

No method of electronic transmission or storage is 100% secure. While we take every reasonable precaution, we cannot guarantee absolute security and encourage you to use a strong, unique password for your account.
